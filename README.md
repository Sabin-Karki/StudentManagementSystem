Student Management System
This Student Management System is a web application designed to help educational institutions manage student information efficiently. It allows users to perform CRUD (Create, Read, Update, Delete) operations on student records stored in a database.

-Features
Create: Add new student records with details like name, email, phone, faculty, and gender.
Read: View existing student information, including name, email, phone, faculty, and gender.
Update: Modify and update student details such as name, email, phone, faculty, and gender.
Delete: Remove student records from the database.

-Technologies Used
PHP: Backend scripting language for server-side logic.
MySQL: Database management system for storing student records.
HTML/CSS: Frontend markup and styling for the user interface.
